Smoke Test Notes - MacBook Apple Silicon M4 Pro Support

Validated in sandbox:
- app/server.py starts successfully with Python runtime available in sandbox.
- GET /api/health returns 200 JSON.
- GET /api/openapi returns 200 JSON with openapi/info/paths.
- POST /api/reliability/reset returns 200 JSON.
- GET /api/reliability/rate-limited returns sequence 200,200,200,429 after reset.
- GET /api/reliability/load-balanced returns backend distribution response.

Not executable in sandbox:
- Docker Desktop for macOS and n8n container runtime cannot be launched from this sandbox environment.

- Browser/Postman on Mac -> use localhost.
- n8n inside Docker -> use host.docker.internal to call the demo app on Mac host.
